# Renew

Marketing-first scaffold for Renew, a stablecoin billing and settlement platform on Avalanche.

## Workspace layout

- `client/` contains the Next.js 16 App Router frontend.
- `server/` is reserved for backend billing APIs later.
- `contracts/` is reserved for Avalanche contract code and ABIs later.

## Frontend stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- TypeScript

## Run the client

```bash
cd client
npm install
npm run dev
```

Use Node.js 20.9.0 or newer.
